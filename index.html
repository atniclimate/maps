Embedding in Squarespace

Basic Embed

Add a Code Block to your Squarespace page and paste the iframe code:

<iframe
  src="https://indigenousaccess.github.io/maps/embed/hazards.html"
  width="100%"
  height="500"
  frameborder="0"
  style="border: none; border-radius: 8px;"
></iframe>

URL Parameters

Customize the map behavior with URL parameters:

Parameter Values Description
region pnw, wa, or, id, mt Initial map region
tribe Tribal code Focus on specific Tribal territory
controls full, minimal, none Control bar visibility

Example: Washington State Focus

<iframe src="embed/hazards.html?region=wa" ...></iframe>

Example: Quinault Territory Focus

<iframe src="embed/tribal-lands.html?tribe=quinault" ...></iframe>
